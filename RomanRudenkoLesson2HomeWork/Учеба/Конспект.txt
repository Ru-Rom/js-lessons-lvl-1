let - аналог var однако виден только в своей области видимости где был создан
const - переменная которая в ходе жизненного цикла программы изменяться не должна, данные переменные не возможно поменять, переназначить
Если используем let и const то перестаем пользоваться старым var

switch:

switch (chtoto) {
		case "ночь":
			console.log("Доброй ночи!");
			break;
		case "утро":
			console.log("Доброе утро!");
			break;
		case "день":
			console.log("Добрый день!");
			break;
		default:
			console.log("Добрый вечер!");
}
